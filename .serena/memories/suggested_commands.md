# よく使うコマンド
## 依存関係
- pnpm install
- npm install
- bun install

## 開発/ビルド
- pnpm dev / npm run dev
- pnpm build / npm run build
- pnpm start / npm run start

## Lint/Format
- pnpm check / npm run check   # Biome lint
- pnpm fix / npm run fix       # Biome auto-fix

## DB (Drizzle)
- pnpm db:generate / npm run db:generate
- pnpm db:migrate / npm run db:migrate
- pnpm db:push / npm run db:push
- pnpm db:studio / npm run db:studio

## Utility
- pnpm create-account / npm run create-account

## Linux基本コマンド
- git, ls, cd, pwd, cat, sed, grep, find
